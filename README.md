# MutatorsMod

Mod for StarCraft II arcade. It expands on Co-op features adding many features. Everything is optional and the base experience is the same as in the normal Co-op, which is useful for testing.

# Notable feautres:

* 16 Additional mutators
* Twitch integration
* Many debug commands useful for testing
* Additional logging and statistics
* UI/UX tweaks (Custom slim UI, Tychus hotkeys, Nydus Worms, Healer AI, top bar notifications)
* Old challenges and mutators unavailbe elsewhere
* Karax commander full rebalance (pre-prestige)
